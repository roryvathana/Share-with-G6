# [CPF: Computer Programming Fundamental](CPF:-Computer-Programming-Fundamental)

  * [GeeksforGeeks](https://www.geeksforgeeks.org/c-programming-language) : The best  platform offering a wealth of resources for computer science education, programming practice, and technical interview preparation.
  * [FreeCodeCamp](https://www.freecodecamp.org/) : Excellent web site for beginner that offer tutorials.
  * [Codecademy](https://www.codecademy.com/): offer fundamental programmer a hand on project
  * [W3Schools](https://www.w3schools.com/) : is a go-to resource, offering tutorials, references, and examples
  * [Stack Overflow](https://stackoverflow.com/) : Community platform where programmer can ask questions, seek advice, and learn from the experiences of others.
